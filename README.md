# Grivn — Discord Moderation & Utility Bot

Grivn is a powerful open-source Discord bot built with `discord.js`, focused on moderation, logging, and utility commands. It’s designed to be lightweight, fast, and fully slash-command based — with no premium restrictions.

---

## 🌟 Features

- 🔨 Moderation: `/warn`, `/kick`, `/ban`, `/mute`, `/infractions`
- 🧹 Message Cleanup: `/purge` with transcript logging
- 🎫 Ticket System: Thread-based support tickets via `/open ticket` and `/close ticket`
- 🎙 Voice Logging: Join/leave events, durations, temp VC logs
- 👋 Welcome & Leave Logging: Custom DMs, invite tracker, time in server
- 👤 Profile Logging: Avatar, username, and nickname changes
- ⚙️ Built using modern slash commands and modular codebase
